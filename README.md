# ASTEROIDS - SFML

Implementación del clásico **Asteroids**, desarrollada en **C++ utilizando SFML**.

El jugador controla una nave espacial que debe destruir asteroides mientras evita colisionar con ellos.  
Los asteroides se fragmentan en partes más pequeñas al ser destruidos.

---

## Tecnologías Utilizadas

- C++
- SFML
- Visual Studio

---

## Descripción del Juego

El objetivo del juego es sobrevivir el mayor tiempo posible destruyendo asteroides y evitando colisiones.

Características principales:

- Los asteroides se fragmentan al ser destruidos
- Los asteroides más pequeños otorgan más puntos
- Sistema de movimiento con rotación de la nave
- Sistema de disparos

---

## Controles

| Tecla | Acción |
|------|------|
| W | Avanzar |
| S | Retroceder |
| A | Girar izquierda |
| D | Girar derecha |
| Espacio | Disparar |
